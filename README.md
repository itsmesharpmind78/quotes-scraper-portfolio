# Quotes Scraper Portfolio

This project scrapes quotes, authors, and tags from [Quotes to Scrape](http://quotes.toscrape.com/js/) dynamic pages.

## Features

- Scrapes **all pages dynamically** using Playwright
- Extracts **Quote, Author, Tags**
- Cleans and deduplicates data (**ETL workflow**)
- Exports to **CSV & Excel**
- Takes **screenshot** for portfolio
- Fully **portfolio-ready** for freelance platforms

## Installation

```bash
pip install pandas playwright openpyxl
playwright install
